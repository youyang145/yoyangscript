# 悠扬工具箱 (YoyangScript)

一个轻量的脚本管理器，可以上传、运行、批量管理你的脚本，并支持自定义背景和多设备同步。以及后面会添加工具箱功能添加一些有趣的工具

> 这是一个**个人学习项目**，代码可能不够专业，但我会慢慢改进。  
> 本项目部分代码由 AI 辅助生成，经过人工测试和调整。

## ✨ 功能
- 📁 上传、删除、批量管理脚本
- ▶️ 后台运行脚本，实时查看日志
- 🖼️ 自定义背景图片，换背景后其他设备自动同步
- 🔔 自动检测 GitHub 更新（需自行配置仓库）
- 📱 响应式界面，手机/电脑均可使用

## 🚀 如何使用
1. 下载并解压本项目。
2. 确保已安装 Python 3.7+。
3. 双击 `悠扬工具箱.bat`（Windows）或在终端执行 `bash 悠扬工具箱.sh`（Mac/Linux）。
4. 按提示允许安装 Flask 库。
5. 浏览器访问 `http://localhost:5000` 即可使用。

详细说明见 `使用说明.txt`。

## 🛠️ 技术栈
- 后端：Python + Flask
- 前端：原生 HTML/CSS/JS
- 脚本执行：Python subprocess（后台线程）
- 更新检测：GitHub API
